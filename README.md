<a name="Overiew"></a>  
## 1. Overview  

**busyman** is a lodash-like JS utility library that provides commonly used functions.  
  
<a name="Installation"></a>
## 2. Installation

> $ npm install busyman --save
  

<a name="Usage"></a>
## 3. Usage

```js
var _ = require('busyman');

console.log(_.isArray('hello'));    // false
```
